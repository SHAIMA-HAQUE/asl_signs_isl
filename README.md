## Setting up this project
``
$ git clone git@github.com:SHAIMA-HAQUE/asl_signs_isl.git
``

``
$ cd asl_signs_isl
``
##### Download the dependencies

``
$ pip install -r requirements.txt
``

- `$ cd holistics`
- `$ python capture_sign.py`
